# Active Directory의 Logical Structure 배경 지식

- Active Directory forest
- Security boundary
- Active Directory domain
    - Network-wide user identity
    - Authentication
    - Trust relationships
    - Replication
- Active Directory organizational units
- Delegation of authority

# Design 시 고려해야 할 일반적인 특징들

- Complexity
- Cost
- Fault Tolerance
- Performance
- Scalability
- Security
